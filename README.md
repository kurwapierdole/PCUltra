# PCUltra - Быстрый старт

## Установка

1. **Установите Python 3.11+**
   - Скачайте с https://www.python.org/downloads/
   - При установке отметьте "Add Python to PATH"

2. **Установите зависимости**
   ```bash
   pip install -r requirements.txt
   ```

3. **Установите браузеры для Playwright**
   ```bash
   playwright install chromium
   ```

## Первый запуск

1. **Запустите приложение**
   ```bash
   python main.py
   ```

2. **Откройте Web UI**
   - Автоматически откроется браузер, или перейдите на http://127.0.0.1:5000
   - При первом запуске откроется страница регистрации:
     * Введите имя пользователя администратора
     * Введите пароль (минимум 6 символов)
     * Подтвердите пароль
   - После регистрации вы автоматически войдете в систему
   - При последующих запусках используйте созданные учетные данные для входа

3. **Настройте бота**
   - Получите токен бота у @BotFather в Telegram
   - Введите токен в разделе "Конфигурация бота"
   - Сохраните конфигурацию

4. **Добавьте себя в whitelist**
   - Узнайте свой Telegram User ID у @userinfobot
   - Добавьте свой ID в разделе "Пользователи"
   - Бот автоматически перезапустится

5. **Используйте бота**
   - Найдите своего бота в Telegram
   - Отправьте `/start`
   - Используйте `/help` для списка команд

## Основные команды

### Управление мышью
- `/mouse_move 100 200` - Переместить мышь на координаты (100, 200)
- `/mouse_click L` - Кликнуть левой кнопкой мыши
- `/scroll 3` - Прокрутить вверх на 3 единицы

### Управление клавиатурой
- `/keyboard Hello World` - Ввести текст
- `/hotkey Win+D` - Нажать Win+D (показать рабочий стол)
- `/hotkey Ctrl+C` - Копировать

### Скриншоты
- `/screenshot` - Полный скриншот экрана
- `/screenshot 100 100 800 600` - Частичный скриншот (x, y, width, height)

### Системные команды
- `/open_app C:\Windows\notepad.exe` - Открыть приложение
- `/open_folder C:\Users` - Открыть папку
- `/notify Привет!` - Показать уведомление

### Браузер
- `/browser_open` - Открыть браузер
- `/browser_navigate https://google.com` - Перейти на сайт
- `/browser_click button#submit` - Кликнуть по элементу

### Shortcuts
- `/vpn` - Запустить AmneziaVPN (настраивается в Web UI)

## Решение проблем

### Бот не запускается
- Проверьте правильность токена
- Убедитесь, что ваш User ID добавлен в whitelist
- Проверьте логи в system tray

### Playwright не работает
- Убедитесь, что браузеры установлены: `playwright install chromium`
- Проверьте, что Playwright имеет права на запуск браузера

### Web UI не открывается
- Проверьте, что порт 5000 свободен
- Убедитесь, что файрвол не блокирует соединение
- Проверьте логи в system tray

### Команды не выполняются
- Убедитесь, что у вас есть права на выполнение команд
- Проверьте, что приложения/пути существуют
- Посмотрите логи бота

## Безопасность

⚠️ **ВАЖНО:**
- Обязательно измените пароль admin после первого входа!
- Добавляйте в whitelist только доверенных пользователей
- Не делитесь токеном бота
- Используйте бота только в доверенной сети

## Поддержка

При возникновении проблем:
1. Проверьте логи в system tray
2. Убедитесь, что все зависимости установлены
3. Проверьте конфигурацию в config.yaml
4. Убедитесь, что у приложения есть необходимые права
